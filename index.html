      Plan format to give an LLM

      Ask for strict JSON in this schema. Exercise names can vary; the app normalizes case, spaces, punctuation, common abbreviations, and creates custom exercises if unmatched.

      Prompt snippet: “Return only valid JSON matching this schema: planName string, notes string optional, days array; each day has day, focus optional, exercises array; each exercise has name required, sets optional number, reps optional string, targetWeightKg optional number/string, restSeconds optional number, notes optional string.”
      Local data & backup
      Your logs, plans, macros, water, and custom exercises are saved locally on this phone using browser localStorage under the stable key hfa-state. App updates should not remove this data while you keep the same installed app/browser origin.
      Tip: export a backup before clearing browser data, changing phones, or reinstalling the app.
      

      Built as a local-first browser app. No LLM generation or AI features are implemented.
