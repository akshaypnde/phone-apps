Data sources and limits

  • Open-Meteo powers forecasts without an API key.
  • transport.rest / Deutsche Bahn-compatible data is used for live rail connection checks when available.
  • Exact bus, rail and flight fares often require vendor booking pages, so the app shows value estimates and opens DB, FlixBus and flight-deal searches.
  • No text-generation service is used inside the app; ranking is deterministic scoring in the browser.
